GTD
Getting Things Done — por David Allen
Este app implementa o método GTD para ajudar você a organizar tudo o que tem a fazer, liberando sua mente para o que realmente importa.
✉
1. Capture — Coloque tudo que está na sua cabeça na Caixa de Entrada
⚙
2. Processe — Use o assistente para decidir o que fazer com cada item
▶
3. Organize — Ações, Projetos, Calendário, Aguardando, Algum Dia
🔎
4. Revise — Faça a Revisão Semanal para manter o sistema atualizado
⚡
5. Execute — Escolha suas próximas ações por contexto, energia e tempo
Dica: Seus dados ficam salvos na nuvem e sincronizados em todos os seus dispositivos. Use Exportar Dados no menu lateral para backup extra.

Importar Tarefas

Faça upload de um arquivo CSV exportado do Todoist. Tarefas concluídas serão ignoradas.

Cole o texto abaixo. Cada linha será adicionada como um item na Caixa de Entrada.

Use convenções para classificar automaticamente:

# Nome do Projeto → Cria um Projeto
- Tarefa simples → Caixa de Entrada
- Tarefa @contexto → Com contexto extraído
- Tarefa [2026-03-10] → Calendário (data)
- Tarefa [2026-03-10 14:00] → Calendário (data+hora)
- Tarefa >> Pessoa → Aguardando (delegada)
~Tarefa ou - ~Tarefa → Algum Dia/Talvez
Linhas sob um # Projeto viram ações desse projeto.
